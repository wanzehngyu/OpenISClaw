# 面板回归操作手册与诊断标准

## 固定效应模型选择指南

### 1. 固定效应层级选择

| 模型 | 适用场景 | 固定效应 |
|------|----------|----------|
| FE (Entity) | 研究对象间差异重要，组内变异足够 | αᵢ |
| FE (Time) | 时间冲击重要（如政策变化），跨期趋势显著 | γₜ |
| TWFE | 标准基准回归，控制个体与时间不可观测异质性 | αᵢ + γₜ |
| TWFE + 双向聚类 | 序列相关与异方差同时存在 | αᵢ + γₜ + εᵢₜ |

### 2. 聚类标准误选择

**为什么需要聚类？**

面板数据中，同一个体在不同年份的误差项通常存在序列相关（Serial Correlation）。若忽略此特征，标准误将被低估，导致 t 值膨胀、过度拒绝原假设。

**聚类维度判断：**
- `cluster_entity`：个体维度聚类（最常用），适用于企业/个人层面的冲击
- `cluster_time`：时间维度聚类，适用于宏观冲击同时影响所有个体
- `cluster_two_way`：双向聚类，同时纠正两种序列相关（最保守）

### 3. VIF 共线性诊断标准

| VIF 范围 | 判定 | 处理建议 |
|----------|------|----------|
| < 3 | 无共线性 | 无需处理 |
| 3 - 5 | 中等共线性 | 密切监控，必要时移除 |
| 5 - 10 | 高度共线性 | 建议移除或合并变量 |
| > 10 | 严重共线性 | 必须移除，否则估计偏误显著 |

**VIF 计算公式：**

$$VIF_j = \frac{1}{1 - R_j^2}$$

其中 $R_j^2$ 是将第 j 个变量对其他解释变量回归的 R²。

### 4. 模型比较准则

#### 4.1 Within R²

$$R^2_{within} = 1 - \frac{\sum_{i}\sum_{t}\hat{\varepsilon}_{it}^2}{\sum_{i}\sum_{t}(Y_{it} - \bar{Y}_i)^2}$$

含义：个体内部变异解释力度。R² > 0.2 为可接受，R² > 0.4 为较好。

#### 4.2 F 统计量（联合显著性检验）

$$F = \frac{(SSR_{restricted} - SSR_{full}) / q}{SSR_{full} / (N - k - 1)}$$

判定：p < 0.05 时拒绝原假设（所有系数同时为 0）。

#### 4.3 信息准则（AIC / BIC）

用于非嵌套模型比较，越小越好：

$$AIC = -2\ln(L) + 2k$$
$$BIC = -2\ln(L) + k\ln(N)$$

### 5. 固定效应去中心化问题

**LSDV vs 组内变换：**

`linearmodels.panel.PanelOLS` 默认使用组内变换（Within Transformation），等价于对变量做去中心化：

$$\tilde{Y}_{it} = Y_{it} - \bar{Y}_i - \bar{Y}_t + \bar{\bar{Y}}$$

**注意事项：**
- 常数项无法估计（LSDV 中 αᵢ 吸收了截距）
- 时间固定效应与个体固定效应同时存在时，需要 `drop_time_effects=True` 避免共线性

### 6. 常见错误与解决方案

| 错误信息 | 原因 | 解决方案 |
|----------|------|----------|
| "entity effect has no within variation" | 变量在组内无变异 | 剔除该变量或改用 LSDV |
| "singular matrix" | 固定效应与变量共线性 | 检查时间趋势变量是否与 γₜ 完全相关 |
| "nobs < neqc" | 样本量不足以估计 FE | 扩充样本或减少固定效应 |
