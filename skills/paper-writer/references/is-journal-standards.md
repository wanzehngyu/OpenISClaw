# IS 期刊格式规范

## 目标期刊参考标准

| 期刊 | 缩写 | 偏好语言 | 审稿周期 | 字数要求（摘要+正文） |
|------|------|---------|---------|---------------------|
| MIS Quarterly | MISQ | 英文 | 6-12 个月 | 约 15000 词 |
| Information Systems Research | ISR | 英文 | 6-12 个月 | 约 15000 词 |
| Journal of the AIS | JAIS | 英文 | 6-12 个月 | 约 15000 词 |
| Journal of Management Information Systems | JMIS | 英文 | 6-9 个月 | 约 12000 词 |
| European Journal of Information Systems | EJIS | 英文/中文 | 6 个月 | 约 10000 词 |
| 信息系统学报 | CJIS | 中文 | 3-6 个月 | 约 10000 词 |

---

## 各期刊摘要格式要求

### MISQ / ISR / JAIS（英文顶刊）
- **字数**：150-250 词
- **结构**：研究问题 → 理论背景 → 研究方法 → 主要发现 → 理论贡献
- **关键词**：5-7 个

### CJIS（中文期刊）
- **字数**：300-500 字
- **结构**：研究问题 → 方法 → 主要结论 → 贡献
- **关键词**：3-5 个

## 引言（Introduction）规范

### IS 顶刊引言结构（英文）

```
第1段：研究背景与问题提出（1段）
  ↓
第2段：文献综述与研究空白（2-3段）
  - 先回顾与本研究直接相关的文献（不要写成文献综述章节）
  - 指出已有研究的不足或争议
  ↓
第3段：研究目的（1段）
  ↓
第4段：研究贡献（1段）
  - 理论贡献（2-3条）
  - 方法贡献（如有）
  - 实践贡献（如有）
  ↓
第5-6段：文章结构预告（1段）
  - "本研究以下内容安排如下：第二节……"
```

### 英文引言写作注意事项
- 第一句话直接切入研究主题，避免宽泛的"随着信息技术的发展……"
- 文献综述不要写成独立章节，要与研究空白自然结合
- 研究贡献要用"this study contributes..."句式，清晰列出

## 理论基础（Theoretical Foundations）规范

### 理论选择原则
1. 理论应能**直接解释**研究现象的机制
2. 选择**主流、成熟**的 IS 理论（便于审稿人理解）
3. 避免在同一篇论文中引入多个竞争性理论（除非研究设计需要对比）

### 常见 IS 理论及其适用场景

| 理论 | 英文 | 典型研究问题 | 推荐期刊 |
|------|------|-------------|---------|
| 资源基础观 | RBV | IT资源如何转化为竞争优势 | MISQ, ISR |
| 动态能力理论 | DCT | 企业如何应对数字化转型 | MISQ, ISR |
| 技术接受模型 | TAM/UTAUT | 用户采纳意愿影响因素 | JMIS, EJIS |
| 制度理论 | Institutional Theory | 监管压力对企业IT采用的影响 | ISR, JAIS |
| IS成功模型 | DeLone & McLean | IS项目成功因素 | JMIS |
| 社会信息处理理论 | SIPT | 远程办公/虚拟团队沟通 | JAIS, MISQ |
| 间断均衡理论 | PET | 技术/政策冲击的突变效应 | ISR |
| 网络效应理论 | Network Effects | 平台用户增长与价值 | ISR |
| 技术可供性理论 | Affordance | 平台功能对行为的塑造 | JAIS |

## 研究假设（Hypotheses）规范

### 假设表述标准

**中文：**
```
[H1] 数字化转型对企业绩效（ROA）有显著正向影响。
```

**英文（APA格式）：**
```
[H1] Digital transformation is positively associated with firm performance (ROA).
```

### 假设数量建议
- 主效应假设：1-2 个
- 异质性/调节效应假设：1-2 个
- 机制/中介效应假设：1-2 个
- 总计：4-8 个假设为宜（过多则难以在单篇论文中充分检验）

---

## 方法论（Methodology）规范

### 实证论文方法章节必备要素

**数据来源**：具体到数据库名称、时间截点、筛选标准
**样本描述**：样本量变化过程（原始→剔除→最终）
**变量定义**：所有变量均需说明测量方式和来源
**模型选择依据**：为什么用这个模型而非其他模型
**内生性处理**：明确说明如何处理内生性（工具变量/滞后项/DID等）

### 稳健性检验清单

| 检验类型 | 说明 |
|---------|------|
| 替换变量 | 用替代指标替换因变量/自变量 |
| 缩尾处理 | 改变缩尾分位（1%→5%或1%→99%） |
| 子样本 | 按行业/规模/地区重新估计 |
| 工具变量 | 用其他工具变量重新估计 |
| 滞后项 | 将自变量滞后1-2期 |
| Bootstrap | 重新抽样估计标准误 |
| 安慰剂检验 | 用随机处理组/随机时间点检验 |

## 结果与分析（Results）规范

### 表格呈现规范

**回归结果表必备列**
- 模型编号（M1, M2, M3……）
- 变量名称（中文或英文）
- 系数（保留3位小数）
- 聚类稳健标准误（括号内）
- 显著性标记（*** ** *）
- R² within / between / overall
- F 统计量 / Wald 统计量
- 样本量 N
- 固定效应说明

**表格注释规范**
- 显著性标记说明（*** p<0.01, ** p<0.05, * p<0.1）
- 标准误类型（聚类稳健标准误，企业层面）
- 数据处理说明（缩尾、缺失值处理）
- 控制变量是否在表中显示

## 讨论（Discussion）规范

### 理论贡献表述模板

```
本研究对[理论名称]做出了以下贡献：

第一，本研究发现了[具体发现]，这一发现拓展了[理论名称]对[研究现象]的解释力。
以往研究主要关注[已有研究主题]，但未充分考虑[本研究独特视角]。

第二，本研究揭示了[机制/调节因素]在[理论名称]中的重要作用，
以往研究通常将其视为[已有处理方式]，本研究提供了新的证据。

第三，本研究将[理论名称]置于[中国制度背景/新的技术情境]下进行检验，
拓展了该理论的外部效度和适用范围。
```

### 实践贡献表述模板

```
本研究的发现对企业管理实践具有以下启示：

首先，本研究结果表明[具体发现]，因此管理者在[具体情境]中应当[具体行动建议]。

其次，本研究揭示了[调节/中介因素]的重要性，提示企业应[具体建议]。

最后，本研究的[中国制度背景]发现表明，在[具体制度环境]下，
企业应特别关注[具体因素]对[研究结果]的影响。
```

## 特别注意事项

### 中国制度背景的特殊性
在中国制度背景下研究 IS 问题，应注意：
- 政府与市场关系（Government-Business Relationship）
- 制度环境异质性（东部 vs 中西部、不同所有制）
- 产业政策与管制（产业基金、补贴、环保政策）
- 数字化转型的中国特色路径

### IS 顶刊审稿偏好
- **MISQ**：重理论贡献，要求理论创新或显著拓展
- **ISR**：重方法严谨性，接受多种实证方法
- **JAIS**：重研究问题的重要性，接受探索性研究
- **JMIS**：重研究问题的实践相关性
